A tag meter define uma medição escalar dentro de um intervalo conhecido ou um valor fracionário. Isso também é conhecido como medidor.
Exemplos: uso do disco, relevância do resultado de uma consulta, etc.
Nota: A tag meter não deve ser usada para indicar progresso (como em uma barra de progresso). Para barras de progresso, use a tag progress.

Dica: Sempre adicione a tag label para melhores práticas de acessibilidade!
