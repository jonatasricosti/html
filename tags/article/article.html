Google Chrome

O Google Chrome é um navegador desenvolvido pelo Google, lançado em 2008. O Chrome é o navegador mais popular do mundo atualmente!

Mozilla Firefox

Mozilla Firefox é um navegador de código aberto desenvolvido pela Mozilla. O Firefox é o segundo navegador mais popular desde janeiro de 2018.

Microsoft Edge

Microsoft Edge é um navegador desenvolvido pela Microsoft, lançado em 2015. O Microsoft Edge substituiu o Internet Explorer.
