A tag datalist especifica uma lista de opções predefinidas para um elemento input.
A tag datalist é usada para fornecer um recurso de "preenchimento automático" para elementos input. Os usuários verão uma lista suspensa de opções predefinidas conforme inserem dados.
O atributo id do elemento datalist deve ser igual ao atributo list do elemento input (isso os vincula).

Nota: A tag datalist não é suportada no Safari 12.0 (ou anterior)
