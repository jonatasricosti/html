A tag track trilha ou faixa especifica trilhas de texto para elementos audio ou video.
Este elemento é usado para especificar legendas, arquivos de legenda ou outros arquivos contendo texto, que devem ser visíveis quando a mídia estiver sendo reproduzida.
As trilhas são formatadas no formato WebVTT (arquivos .vtt).

Nota: Esse exemplo foi testado no Live Server do Visual Studio Code
